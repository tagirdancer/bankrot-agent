ШАГ 1 — Установите Python
─────────────────────────
Скачайте с сайта: https://python.org/downloads
Установите как обычную программу.
ВАЖНО: при установке поставьте галочку "Add to PATH"

ШАГ 2 — Заполните .env файл
────────────────────────────
1. Скопируйте файл .env.example
2. Переименуйте копию в .env
3. Откройте .env в блокноте
4. Вставьте свои данные вместо заглушек

ШАГ 3 — Установите зависимости
───────────────────────────────
Откройте командную строку в папке с файлами:
(правая кнопка мыши на папке → Открыть в терминале)

Введите и нажмите Enter:
  pip install -r requirements.txt

Потом:
  playwright install chromium

ШАГ 4 — Запуск
──────────────
Тестовый запуск (прямо сейчас):
  python agent.py --now

Запуск по расписанию (каждый день в 8:00):
  python agent.py
